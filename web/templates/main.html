This tool helps generate food category predictions based on the Concept Unique Identifier (CUI) of a substance. You can find the CUI for a substance on the NCI Metathesaurus website. For example, the CUI for "oat" is C0028753.

Note: It might take several seconds to return the result, as 1) the database is rather large and 2) the model is actively computing the predictions.
