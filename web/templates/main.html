Welcome.


This tool uses neural networks to generate food category predictions based on the Concept Unique Identifier (CUI) of an ingredient or adulterant. You can search for the CUI of a substance on the NCI Metathesaurus website. For example, the CUI for "oat" is C0028753. See the Github repo for more information on how to use this tool and to download the source code.
